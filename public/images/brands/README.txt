BRAND LOGOS — drop the official logo files here.

The homepage "Brands & Equipment Serviced" strip shows each brand's logo, and
falls back to the plain text name until the file exists. To turn a name into a
logo, add the matching file below (paths are set in src/data/services.js →
brandsServiced):

  wabash.svg
  utility.svg
  great-dane.svg
  hyundai-translead.svg
  thermo-king.svg
  carrier-transicold.svg
  stoughton.svg

Notes:
- SVG or transparent PNG. White / monochrome versions look best on the dark
  background. If you use PNG, change the extension in src/data/services.js to
  match (e.g. wabash.png).
- These are third-party trademarks — use each brand's official press / media-kit
  logo. Don't recreate or approximate them.
- Height renders at ~32–40px; provide reasonably high-res assets.
